# Quiz Master Pro - Interactive Quiz Game

A modern and interactive quiz game application developed as part of my Web Development Internship. This project focuses on creating an engaging learning experience with dynamic questions, timers, score tracking, animations, and responsive user interfaces using HTML, CSS, and JavaScript.

## 🚀 Features

* Interactive Multiple Choice Questions
* 20+ HTML, CSS, and JavaScript Questions
* Question Shuffle Functionality
* Countdown Timer for Each Question
* Progress Bar Tracking
* Real-Time Score Calculation
* Answer Explanation After Each Question
* High Score Storage using Local Storage
* Performance Rating System
* Dark / Light Theme Toggle
* Glassmorphism UI Design
* Animated Background Effects
* Confetti Celebration on Completion
* Responsive Design for Mobile, Tablet, and Desktop
* Restart Quiz Functionality
* Smooth Animations and Hover Effects
